# Pronunciation Mistake Detection App

A speech-based pronunciation evaluation system using:
- Speech Recognition
- Phoneme analysis
- Flask backend
- HTML/JS frontend

## Features
✓ speech-to-text
✓ phoneme comparison
✓ pronunciation feedback
✓ word-level evaluation
✓ API endpoints

## API Routes
GET /get-sentence
POST /evaluate

## Tech Stack
Python, Flask, CMUdict, Librosa, SpeechRecognition
